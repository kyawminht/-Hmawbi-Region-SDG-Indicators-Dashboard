## 🎯 The Challenge

My sister (Public Health Dept) needed to present SDG data for **6 health facilities**. Tables weren't engaging enough.

**The problem:** I didn't know how to build a clickable map with coordinates.

**The solution:** Found [Image-Map.net](https://www.image-map.net/) — drew polygons, got coordinates, embedded in `<svg>` `<polygon>` tags.
